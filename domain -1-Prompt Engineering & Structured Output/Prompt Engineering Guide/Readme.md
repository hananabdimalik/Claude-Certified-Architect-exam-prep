# Details and Explanation of the python files in the folder.

**001_requests.ipynb**
This file will guide you on installing your dependencies, making your first request to Claude API. 
There is also a Chatbot that accepts input from user, try it out! This will help you to understand how to create a chat and use it to get response from Claude.

**002_system_prompt.ipynb**
This file will guide you on how to create system prompt and use it to format the Output response.

**system_prompt_exercise.ipynb**
This file is all about system prompt and an exercise that will help you to understand system prompt very well.

**003_temperature.ipynb**
This file will guide you on how to add temperature and also iterate it to get your desired output response.

**004_streaming.ipynb**
This file will guide you on how to add stream to the message manually and also by using the SDK's simplified streaming interface that extracts just the text content.

**001_prompt_evals.ipynb**
This file will guide you on how to generate dataset and the evaluation pipeline which represents the foundation of most AI evaluation systems.

**001_prompt_evals_grader.ipynb**
This file will guide you on how to create the grading system and add it to your evaluation workflow.

**001_prompt_evals_fns.ipynb**
This file will guide you on how to use pre-filled assistant message to run the prompt to improve prompt clarity and how to merge the model grader score with the code grader score divided by 2 (average).

**002_prompting.ipynb**
This file is a complete guide to understanding the different prompt engineering techniques, try them out! to see how it increases the prompt's quality with a good output response.
It will also guide you on how to write a good quality prompt by using the Iterative Improvement Process. 

**dataset.json**
This is the generated dataset containing the different test cases prompt inputs, descrption and also solution criteria which were generated from the prompt. This is for the one-day meal plan for an athlete that meets their dietary restrictions example.
